== Introduction

Versioned implements the Semantic Versioning Standard
footnote:semver[SemVer 2.0.0 (https://semver.org/spec/v2.0.0.html)]
and general version number concepts. Although SemVer is the first and main
version format supported, it is not intended to be the last. The concepts in
this library are meant to be reusable to different formats. And hopefully to
provide for transforming between such different versioning formats.
